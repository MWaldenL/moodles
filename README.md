## Tech Stack:
1. Texture
2. RxSwift and RxCocoa
3. ReSwift
4. Node.js
5. MongoDB

## Running the app
Open a terminal in the project folder and type ```pod install```. Then, build the project in XCode and run the app in a simulator.
